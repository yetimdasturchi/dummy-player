# Dummy video player

Simple video player built with **SDL2** and **FFmpeg**. 

### Install dependencies

```bash
sudo apt update
sudo apt install \
  libsdl2-dev libsdl2-ttf-dev \
  libavformat-dev libavcodec-dev libavutil-dev \
  libswscale-dev libswresample-dev
```
## Build & Run

```
make clean
make
./player
```
